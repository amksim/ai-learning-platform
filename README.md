## Запуск проекта

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000)

## 🚀 Быстрый деплой

### Вариант 1: Vercel CLI (рекомендуется)

```bash
# Установи Vercel CLI
npm install -g vercel

# Деплой
npx vercel

# Следуй инструкциям в терминале
```

### Вариант 2: GitHub + Vercel Dashboard

1. **Загрузи на GitHub:**
```bash
git init
git add .
git commit -m "Initial commit"
git push origin main
```

2. **Подключи Vercel:**
   - Зайди на [vercel.com](https://vercel.com)
   - Import Repository
   - Выбери свой репозиторий
   - Deploy! 🎉

### Переменные окружения

Добавь в Vercel Dashboard:
```env
NEXTAUTH_URL=https://your-domain.vercel.app
NEXTAUTH_SECRET=your-random-secret
NODE_ENV=production
```

### После деплоя

✅ Сайт работает на `your-app.vercel.app`  
✅ Все функции доступны  
✅ Платежи в режиме симуляции  
✅ Можно подключить свой домен  

**Подробная инструкция:** см. [DEPLOY.md](DEPLOY.md)
